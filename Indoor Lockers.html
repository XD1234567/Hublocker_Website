Software Requirement

  • API Requirements to control the locker we need your API to be able to create our software that can control the locks and motherboard of the locker. The API should be able to provide the following;
    1. Ability to query the Locker for information link slots that are open and slots that are closed.
    2. Ability to know if a package in a slot (sensor).
  • Operating system requirement
    1. The operation system of the locker should be preferably andriod. if andriod is not available, then, other OS is fine.
    2. High processing operating system should be used to avoid delay in response time.
